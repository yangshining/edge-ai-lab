Voice AI Assistant Concept

The ESP32-S3 becomes a voice terminal. The backend owns the model API key and speech pipeline.

Device UI: Digital Avatar
^_^
Listening / Thinking / Speaking
Tiny captions: optional
Recommended Data Flow
1

Wake

Button first, keyword later if hardware and memory allow.

2

Record

I2S mic captures short audio and streams/uploads to your backend.

3

Think

Backend runs ASR, calls the configured LLM, then runs TTS.

4

Speak

ESP32 plays response audio while the avatar animates.
