# STM32F103 智能传感器监控系统 - 项目介绍与教程

## 项目概述

本项目是一个基于STM32F103C8T6微控制器的智能传感器监控系统，集成了多种环境传感器（温湿度、光照、气体浓度等），通过ESP8266 WiFi模块实现物联网连接，将传感器数据上传至OneNET物联网平台，并具备本地显示、自动控制和报警功能。

### 主要特性

- **多传感器支持**：DHT11温湿度传感器、MQ系列气体传感器（MQ2-MQ7）、光照传感器
- **物联网接入**：通过ESP8266 WiFi模块，使用MQTT协议连接OneNET平台
- **本地显示**：OLED显示屏实时显示传感器数据
- **自动控制**：根据阈值自动控制风扇、补光灯、蜂鸣器
- **数据上传**：定时将传感器数据上传至云端平台

DHT11通过单总线协议通信，需要精确的时序控制：

```c
//复位DHT11
void DHT11_Rst(void)	   
{                 
	DHT11_IO_OUT(); 	//SET OUTPUT
	DHT11_DQ_OUT(0); 	//拉低DQ
	DelayXms(20);    	//拉低至少18ms
	DHT11_DQ_OUT(1); 	//DQ=1 
	DelayUs(30);     	//主机拉高20~40us
}

//等待DHT11的响应
//返回1:未检测到DHT11的存在
//返回0:存在
uint8_t DHT11_Check(void) 	   
{   
	uint8_t retry=0;
	DHT11_IO_IN();//SET INPUT	 
	while (DHT11_DQ_IN&&retry<100)//DHT11会拉低40~80us
	{
		retry++;
		DelayUs(1);
	};	 
	if(retry>=100)return 1;
	else retry=0;
	while (!DHT11_DQ_IN&&retry<100)//DHT11拉低后会再次拉高40~80us
	{
		retry++;
		DelayUs(1);
	};
	if(retry>=100)return 1;	    
	return 0;
}

//从DHT11读取一个位
//返回值：1/0
uint8_t DHT11_Read_Bit(void) 			 
{
	uint8_t retry=0;
	while(DHT11_DQ_IN&&retry<100)//等待变为低电平
	{
		retry++;
		DelayUs(1);
	}
	retry=0;
	while(!DHT11_DQ_IN&&retry<100)//等待变高电平
	{
		retry++;
		DelayUs(1);
	}
	DelayUs(40);//等待40us
	if(DHT11_DQ_IN)return 1;
	else return 0;		   
}

//从DHT11读取一个字节
//返回值：读到的数据
uint8_t DHT11_Read_Byte(void)    
{        
	uint8_t i,dat;
	dat=0;
	for (i=0;i<8;i++) 
	{
		dat<<=1; 
		dat|=DHT11_Read_Bit();
	}						    
	return dat;
}
```

#### ADC多通道采集

系统使用DMA方式实现多通道ADC采集，提高效率：

```12:80:hardware/src/adc.c
static void ADCx_GPIO_Config(void)
{
	GPIO_InitTypeDef GPIO_InitStructure;
	
	// 打开 ADC IO端口时钟
	ADC_GPIO_APBxClock_FUN ( ADC_GPIO_CLK, ENABLE );
	
	// 配置 ADC IO 引脚模式 (PA0-PA7, 8 channels)
	GPIO_InitStructure.GPIO_Pin = ADC_PIN0 | ADC_PIN1 | ADC_PIN4 | ADC_PIN5 | ADC_PIN6 | ADC_PIN7;
	GPIO_InitStructure.GPIO_Mode = GPIO_Mode_AIN;
	// 初始化 ADC IO
	GPIO_Init(ADC_PORT, &GPIO_InitStructure);
}

/**
  * @brief  配置ADC工作模式
  * @param  无
  * @retval 无
  */
static void ADCx_Mode_Config(void)
{
	DMA_InitTypeDef DMA_InitStructure;
	ADC_InitTypeDef ADC_InitStructure;
	
	// 打开DMA时钟
	RCC_AHBPeriphClockCmd(ADC_DMA_CLK, ENABLE);
	// 打开ADC时钟
	ADC_APBxClock_FUN ( ADC_CLK, ENABLE );
	
	// 复位DMA控制器
	DMA_DeInit(ADC_DMA_CHANNEL);
	
	// 配置 DMA 初始化结构体
	// 外设基址为：ADC 数据寄存器地址
	DMA_InitStructure.DMA_PeripheralBaseAddr = ( u32 ) ( & ( ADC_x->DR ) );
	
	// 存储器地址
	DMA_InitStructure.DMA_MemoryBaseAddr = (u32)ADC_ConvertedValue;
	
	// 数据源来自外设
	DMA_InitStructure.DMA_DIR = DMA_DIR_PeripheralSRC;
	
	// 缓冲区大小，应该等于数据目的地的大小
	DMA_InitStructure.DMA_BufferSize = NOFCHANEL;
	
	// 外设寄存器只有一个，地址不用递增
	DMA_InitStructure.DMA_PeripheralInc = DMA_PeripheralInc_Disable;

	// 存储器地址递增
	DMA_InitStructure.DMA_MemoryInc = DMA_MemoryInc_Enable; 
	
	// 外设数据大小为半字，即两个字节
	DMA_InitStructure.DMA_PeripheralDataSize = DMA_PeripheralDataSize_HalfWord;
	
	// 内存数据大小也为半字，跟外设数据大小相同
	DMA_InitStructure.DMA_MemoryDataSize = DMA_MemoryDataSize_HalfWord;
	
	// 循环传输模式
	DMA_InitStructure.DMA_Mode = DMA_Mode_Circular;	

	// DMA 传输通道优先级为高，当使用一个DMA通道时，优先级设置不影响
	DMA_InitStructure.DMA_Priority = DMA_Priority_High;
	
	// 禁止存储器到存储器模式，因为是从外设到存储器
	DMA_InitStructure.DMA_M2M = DMA_M2M_Disable;
	
	// 初始化DMA
	DMA_Init(ADC_DMA_CHANNEL, &DMA_InitStructure);
```

## 功能说明

### 自动控制逻辑

系统根据预设阈值自动控制执行器：

- **温度报警**：温度 ≥ 37°C 时，开启风扇并报警
- **湿度报警**：湿度 ≥ 90% 时，开启风扇并报警
- **光照控制**：光照 ≤ 30% 时，开启LED补光灯
- **气体报警**：MQ2传感器值 ≥ 50 时，触发蜂鸣器报警

### 数据上传格式

```json
  "id": "123",
  "params": {
    "temp": {"value": 25},
    "humi": {"value": 60},
    "smog1": {"value": 45},
    "beep": {"value": false}
  }
```


## 常见问题

### 1. ESP8266连接失败

- 检查ESP8266的电源和接线
- 确认ESP8266已连接到WiFi网络
- 检查USART2的波特率配置（115200）

### 2. OneNET平台连接失败

- 检查产品ID和访问密钥是否正确
- 确认网络连接正常
- 查看串口调试信息获取详细错误

### 3. 传感器数据异常

- 检查传感器接线是否正确
- 确认传感器供电正常
- 检查ADC配置和DMA传输是否正常

### 4. OLED显示异常

- 检查I2C接线（SDA、SCL）
- 确认OLED地址配置正确
- 检查I2C初始化代码
