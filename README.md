# WhyNotAI

> A collection of hands-on experiments exploring AI on embedded and edge devices, including ESP32, STM32, and lightweight AI agents.

---

## 🚀 Overview

This repository is an experimental playground for building and exploring AI systems on resource-constrained hardware.

The goal is simple:

- Run AI beyond the cloud
- Bring intelligence to microcontrollers
- Explore the boundary between embedded systems and AI agents

---

## 🧠 What’s Inside

This repo focuses on practical and interesting projects such as:

- ⚡ ESP32-based AI agents
- 🔧 STM32 embedded AI demos
- 🤖 Lightweight agent systems on edge devices
- 📡 Local inference, tool calling, and automation
- 🧪 Experiments with TinyML and on-device intelligence
